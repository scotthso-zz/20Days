Day 0: Setting up this site

April 25th 2014

Setting up everything for the site took me a whole day, although it was decently straight forward.

My website was hosted on my friend's server at first and I decided it was time for me to get my own.

-> I used digital ocean as my server
-> For the image I used the latest version of Ubuntu.
-> To start off I installed LAMP onto my VPS
-> I directed my domain servers on godaddy from my friend's to digital oceans'
-> Then I installed a FTP called Filezilla
-> Connected to my host server through Filezilla then dragged my HTML and CSS files into the directory /var/www/html/ (took longest to figure out...)
-> done.

I encountered so many problems. At first I kept on installing and doing things that I actually didn't need which ended making things more complicated than neccessary. Then when I got to the FTP it took me a while to figure out how to connect to the server and which directory to drag my files to.

Lesson is to read the instructions slowly and don't rush.
